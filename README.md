# Calculator Web Application

A responsive calculator built with HTML, CSS, and JavaScript that evaluates mathematical expressions without using JavaScript’s built-in eval() function. The application converts infix expressions to postfix notation using the Shunting Yard Algorithm and evaluates them with a stack-based postfix evaluation algorithm.

✨ Features

* ➕ Basic arithmetic operations (+, -, *, /, %)
* 🔢 Multi-digit and decimal number support
* 📐 Operator precedence and parentheses handling
* 📚 Custom infix-to-postfix conversion
* 🗂️ Stack-based postfix expression evaluation
* ⚠️ Error handling for invalid expressions and division by zero
* 📱 Responsive and user-friendly interface

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

🧠 Algorithms & Data Structures

* Stack
* Tokenization
* Shunting Yard Algorithm
* Postfix Expression Evaluation
* DOM Manipulation
* Event Handling
